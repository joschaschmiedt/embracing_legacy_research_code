Embracing Legacy Research Code | deRSE 2023

Example: VStim

Why was the software written?

To control dynamic psychophysical experiments involving accurate visual stimulation with morphing shapes


Why does it need to change?

Designed for OS that does not do too much multi-tasking, loses accuracy on modern Windows, new features

 

What does the code do and how?

  • Few, stable dependencies: DirectX 7, Microsoft Foundation Classes
  • C++98: raw pointers, not using standard library
  • grown, but relatively modular architecture, cohesive classes
  • documentation as Word files
  • manual tests only
